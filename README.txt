== Description ==

A simple plugin that creates a widget which performs a lightbox for any youtube video.
Just choose an image, a title, a youtube ID and you are good to go.

this plugin does not include any desgin (css), this allow the widget to inherit the look of your theme
 - you may use the class of the widget to alter the desgin in your css file.

== Installation ==

1. Upload the plugin to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Navigate to Appearance > Widgets
4. Drop the widget on your side and continue!


== Frequently Asked Questions ==

1. Does it support RTL?

Yes

2. Can I put more then 1 widget in one page?

Yes,  as many as you like

3. How do I set the image?

By click on the image input box, you get a image uploader.
pick an image and hit the "insert to post" button.

4. Why the image upload does not work

It does not work when you first drag it to the widget area,
you need to click on save and then it will work
